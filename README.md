# Stew

**Stew** is an AI-native personal finance companion for urban Indian Gen Z/millennials. Think: India's Cleo, but smarter, more open, and built for the way Indians actually deal with money.

## Tech Stack
- Next.js 16 (App Router)
- React
- Typescript
- Vanilla CSS + CSS Modules
- `@hugeicons/react`

## Features
- **Persona Engine**: Adapts personality to the user (Fresher or Investor).
- **UPI Intelligence**: Reads and categorizes UPI transactions via RBI Account Aggregator.
- **Salary Day Mode**: Auto-suggests budget splits (Needs, Wants, Savings, Investments) when salary hits.
- **Stew Marketplace**: Community-built AI financial personas plugins (Tax Season Mode, DeFi Degen, etc).
- **Stew Float**: Salary-backed micro-credit for the tight weeks.
- **Goals Tracking**: Track savings targets visually.
